## 🚀 使用说明

### 前置要求
- Node.js 18.0+ 
- npm 或 yarn

### 安装和运行
```bash
# 1. 安装依赖
npm install
cd frontend && npm install
cd ../backend && npm install
cd ..

# 2. 构建后端
npm run backend:build

# 3. 开发模式运行
npm run dev

# 4. 桌面应用
npm run electron:dev

# 5. 生产构建
npm run build
npm run dist
```

### 主要功能操作流程
1. **创建任务**: 首页 → 新建任务 → 填写信息
2. **执行流程**: 待办事项 → 点击执行 → 按步骤操作
3. **管理配置**: 配置管理 → 服务配置 → 添加/编辑服务
4. **查看历史**: 历史记录 → 查看详情 → 导出数据
